# Rock-Paper-Scissors Game

A simple Rock-Paper-Scissors game implemented in **Python** and **JavaScript**.  
This project demonstrates basic programming concepts like functions, conditionals, user input, and random choices.

## Features

- Play against the computer
- Python and JavaScript versions
- Clear game logic using functions
- User-friendly prompts and outputs

## How to Run

### Python Version

1. Make sure you have Python installed (>=3.6)
2. Open terminal/command prompt
3. Navigate to the project folder
4. Run:
```bash
python rps.py
